$x$ 在模 $p$ 意义下存在逆元，当且仅当 $x$ 与 $p$ 互质。

板子（exgcd）：

```cpp
void exgcd(int a, int b, int &x, int &y);
int inv(int a, int p) {
    int x, y;
    exgcd(a, p, x, y);
    return x;
}
```

费马小定理：

当 $p$ 为质数，

若 $a$ 为 $p$ 的倍数，则

$$
a^p\equiv a\pmod{p}
$$

若 $a$ 不为 $p$ 的倍数，则

$$
a^{p-1}\equiv 1\pmod{p}
$$

费马小定理求逆元板子：

```cpp
int inv(int a, int p) {
    return qpow(a, p - 2);
}
```
